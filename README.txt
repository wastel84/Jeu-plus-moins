****************************************
**********JEU DU PLUS OU MOINS**********
****************************************

Voici le repository GIT d'un petit jeu en javascript. Les règles sont :

- Les deux joueurs saisissent leurs noms

- Le joueur 1 saisira en premier le nombre que le joueur 2 devra deviner.

- Lorsque le joueur 2 a deviné le nombre, son nombre d'essais est affiché.

- Puis le joueur 2 saisira le nombre que le joueur 1 devra deviner, et ainsi de suite.
